# QUICK START GUIDE
# Military Alumni Directory - Next.js Edition

## ✅ What's been done

Your project has been successfully migrated from Vue.js + Express to Next.js 14.
All functionality remains the same, just with better architecture and performance.

## 🚀 Getting Started (5 Minutes)

### Step 1: Install Dependencies
```bash
npm install
copy .env.example .env.local
```

### Step 2: Verify PostgreSQL Setup
Make sure PostgreSQL is running on localhost:5432 with:
- Username: postgres
- Password: postgres

Create the database:
```bash
createdb -U postgres crma42
```

### Step 3: Initialize Database
```bash
npm run init-db
```
This will create all tables and seed 20 sample records.

### Step 4: Start Development Server
```bash
npm run dev
```

### Step 5: Open in Browser
Go to: http://localhost:3000

## 📱 Test Credentials

After running `init-db`, you can:
1. Register a new account on the Registration page
2. Account will be pending admin approval until approved
3. An admin account doesn't exist by default - create one via SQL:
```sql
INSERT INTO users (id, email, password_hash, first_name, last_name, role, status)
VALUES (gen_random_uuid(), 'admin@test.com', '$2a$10$...', 'Admin', 'User', 'admin', 'approved');
```

Or just register normally and approve yourself via the admin panel after fixing role manually in DB.

## 🐛 Troubleshooting

### Port 3000 already in use?
```bash
npm run dev -- -p 3001
```

### PostgreSQL connection error?
Check `.env.local` and verify PostgreSQL credentials:
```bash
psql -U postgres -h localhost -d crma42
```

### Database tables not found?
Run initialization again:
```bash
npm run init-db
```

### Clear Next.js cache?
```bash
rm -rf .next
npm run dev
```
